# MedVibe

MedVibe is an iOS app for medication and habit reminders with a full daily history log.

## Features
- One-time and repeating reminders
- Safe SwiftData architecture (no detached crashes)
- Daily history (Completed / Skipped / Missed / Paused)
- Notification handling (complete, snooze, skip)
- 30-day rolling history retention
- Clean SwiftUI architecture with feature-based folders

## Tech Stack
- SwiftUI
- SwiftData
- UserNotifications
- MV-style ViewModels with primitive rows
- Safe re-fetch pattern with PersistentIdentifier

## Status
Actively developed. Core reminder + history system is production-ready.
